/****** BACK END REST API ******//
**********************************
1. open a command prompt
2. go to the jar level
3. execute the command : java -jar bunny_studio_test-0.0.1-SNAPSHOT.jar
The rest API will be started on http://localhost:8080/users-rest/api/v1


/****** BACK END REST API === FOR DEVELOPER ==== ******//
********************************************************
1. get the source code = 
2. run mvn clean  (add -DskipTests to skip run tests)
3. run mvn install (add -DskipTests to skip run tests)
The rest API will started on http://localhost:8080/users-rest/api/v1


/****** FRONT END UI ******//
********************************************************
1. Get the dist folder
2. open a command prompt and go to the jar level of dist/bunny_studio_test
3. execute the command : ng serve OR npm start
The Application will started on http://localhost:4200/



/****** FRONT END UI === FOR DEVELOPER ==== ******//
********************************************************
1. Get the source code
2. run npm install
3. run npm start
The Application will started on http://localhost:4200/
